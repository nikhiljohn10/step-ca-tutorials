# Installation

## Pacman/Arch Linux

```
pacman -S step-cli step-ca
```

## Brew/MacOS

```
brew install step
```

## Kubernetes

```
helm install step-certificates
```

## Alpine Linux

```
apk add step-cli step-certificates
```

## Ubuntu/Debian

```
sudo bash scripts/install.sh
```

For uninstallation, use the following command:
```
sudo bash scripts/uninstall.sh
```
