# Installation
## Pacman/Arch Linux

```
pacman -S step-cli step-ca
```

## Brew/MacOS

```
brew install step
```

## Kubernetes

```
helm install step-certificates
```

## Alpine Linux

```
apk add step-cli step-certificates
```

## Ubuntu/Debian

```
curl -o- https://raw.githubusercontent.com/nikhiljohn10/step-ca-tutorials/main/install_step.sh | sudo bash
```
